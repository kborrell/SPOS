A new cloud service for solving numerical linear optimization problems

New session Check Results

Launch solvers executions

This service allows you to solve your optimization problems on the cloud without worrying about solvers configurations or required hardware. Once the solver execution finishes, you will be able to check and download the results.

Customize your execution

You can customize your execution by choosing the type of model, the solver to be used and the format of model files. And if this is not enough, you will be able to customize the virtual machine where the solver will be running too!

Easy, simple, powerful

All this facilities are offered to you through a simple and easy user interface. A step-by-step process will guide you while creating a new session and you will get a summarised version of the result, of possible. Keep it simple!
